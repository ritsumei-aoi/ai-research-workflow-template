# Prompt Pack: Issue (Optional)

Optional retrospective and check prompts.
Intended for use after the squash merge to main is complete.

## Prompt R1: Resource Check (Optional)

```markdown
Work is now complete.
For confirmation, please present the information you used in this thread.

**Required**:
- The model name currently in use
- A summary of the initial instructions you received

**Optional (only if available)**:
- Number of dialog turns
- Estimated token consumption

If any information is unavailable, explicitly state "Not available."
```

## Prompt R2: Session Retrospective (Optional)

```markdown
Please perform a retrospective of this session objectively and rigorously.
From both the human and AI perspectives, address the following:

- Each party's role
- What went well
- Areas for improvement
- Specific improvement actions for next time (up to 3 items)

Be factual and candid — no hedging.
```
