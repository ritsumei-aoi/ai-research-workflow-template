# Prompt Pack: Review (Optional)

任意で使う振り返り・確認プロンプト群です。
mainへの squash merge が終わった後に用いることを想定しています．

## Prompt R1: Resource Check (Optional)

```markdown
これで作業は完了しました．
確認のため、このスレッドで利用した情報を提示してください。

**必須**:
- 現在使用しているモデル名
- 現在あなたが把握している、こちらの最初の指示内容の要約

**任意（取得可能な場合のみ）**:
- 対話回数
- トークン消費の概算

取得できない情報は「取得不可」と明記してください。
```

## Prompt R2: Session Retrospective (Optional)

```markdown
このセッションの振り返りを、客観的かつ厳密に行ってください。
人間側とAI側の両方の視点から、以下の点について述べてください。

- 各自の役割
- 良かった点
- 改善余地がある点
- 次回の具体的改善アクション（3項目以内）

忖度せず、事実ベースで記述してください。
```
