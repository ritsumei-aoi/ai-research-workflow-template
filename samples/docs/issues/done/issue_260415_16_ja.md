## I16-1. 対象分野への幾何学の導入

### 背景

MathQForm の対象分野として DE（微分方程式）、PS（確率・統計）、RA（実解析）、AA（抽象代数）の4分野を設定していた。
数学の主要分野を網羅するために、幾何学の追加を検討する必要がある。
GT（幾何学・位相幾何学）を追加すれば5分野となり、数学の各領域を網羅的に扱う構成となる。

### 要求

- 現在の4分野構成に幾何学を追加する場合の影響を分析する
- 追加が妥当な場合、スキーマ・計画文書を修正する
  - 5分野の場合、各 100 Item（計 500 Item）に再配分

#### 成果物形式

文書 / スキーマ

#### 関連issue

I16-2（数学分野ラベル付けの再検証）

### 完了条件

- [x] 現在の分野構成と幾何学追加の影響を分析する
- [x] スキーマを5分野対応に更新する
- [x] 計画文書を5分野構成に修正する

---

### 対応

GT（geometry_topology: 幾何学・位相幾何学）を第5分野として導入した。

**分野構成**: DE, PS, RA, AA, GT の5分野（各 100 Item、計 500 Item）
- 予算中立: 125×4 → 100×5 = 500 Item（総数変更なし）
- B4–M1レベル: 点集合位相、曲線・曲面の微分幾何学、代数的位相幾何学入門を対象

**更新ファイル**:
- スキーマ 6ファイル: field enum、ID pattern に GT を追加
- 計画文書 9ファイル: 分野リスト、Item 配分、予算内訳を5分野に更新
