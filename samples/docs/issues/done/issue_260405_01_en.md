## I1. Web Presentation Methods for Code Execution Examples

### Background

The code used in the paper is published in the `osp-triviality` repository.
To allow readers to experience interactive computations, we need to investigate
suitable methods for presenting code execution examples on the Web.
A self-managed web server is available.

### Requirements

Investigate methods for presenting code execution examples on the Web.
Provide a comparison table including advantages, disadvantages, and recommendations.
No implementation is required (proposals only).

### Completion Criteria

- [x] At least 3 methods are compared
- [x] Each method has evaluations for setup effort, interactivity, and maintenance cost
- [x] A recommended configuration (Phase 1 / Phase 2) is specified

---
