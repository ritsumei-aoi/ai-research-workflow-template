## I1. 論文 Introduction の方向性候補の提示

### 背景

英語論文 `docs/theory/triviality_B0n.tex` には現在 Introduction セクションがない。
本論文は Bakalov–Sullivan (arXiv:1612.09400v2) を出発点としており、
Lie 超代数の変形理論の文脈に位置づける必要がある。

### 要求

Introduction として記載する内容の候補を複数提示すること。
各候補について構成案と利点を明記すること。

### 完了条件

- [x] 3 つ以上の方向性候補が提示されている
- [x] 各候補に構成案（項目列挙）と利点が記載されている
- [x] 推奨案が明記されている

---

### 対応

3 つの候補を提示:

**候補 A（推奨）: 変形理論を中心に据えたアプローチ**
- 構成: Lie 超代数の変形理論 → 非斉次双線形形式 → 主定理の概略 → 証明手法（certificate） → 論文構成
- 利点: 数学的動機が明確。変形理論（コホモロジー理論）の文脈に位置づけることで読者層が広がる

**候補 B: 非斉次双線形形式の分類問題中心**
- 構成: 振動子超代数の構成 → 非斉次形式の分類 → 分類から変形へ → 主定理
- 利点: [BS17] との直接的連続性が明確

**候補 C: 計算的証明手法を前面に出す**
- 構成: 計算代数の手法概観 → 背景と問題設定 → 主定理 → 手法の独自性 → 計算的検証設計
- 利点: 計算代数コミュニティへの訴求。公開リポジトリとの整合性

**推奨**: 候補 A。候補 B/C の要素は候補 A 内に統合可能。
