## I05-1. 研究・教育効果の体系的検証

### 背景

MathQForm ベンチマークの設計が進み、教育的・研究的効果を具体化する段階に入った。
SPReAD 申請における主張の優先順位を明確にし、ギャップ分析を行う必要がある。
また、パイロット分野の再選定が課題となっている。

### 要求

- 教育効果・研究効果を体系的に整理する
- 申請主張の優先順位付けとギャップ分析を行う
- パイロット分野の妥当性を再検証する

### 完了条件

- [x] 教育的効果と研究的効果を各5項目以上で整理する
- [x] 12項目を5段階で優先順位付けする
- [x] ギャップ分析に基づく修正提案を提示する

---

### 対応

**教育的効果**（5項目）:
1. 問い形成スキルの明示化
2. AI 足場かけによる学習支援
3. ゼミ実践の改善
4. テーマ設定支援
5. 成熟度の可視化

**研究的効果**（5項目）:
1. ベンチマーク創出
2. 評価方法論の提案
3. 非形式→形式変換の分析
4. 3パス比較による生成品質研究
5. オープンデータの提供

**重要な計画修正**:
- パイロット分野を abstract_algebra → probability_statistics に変更
  - 理由: 他分野展開の説得力、実験科学への波及効果
- 12項目の優先順位付けにて S10（他分野展開）に +3 ギャップを特定 → 4件の修正提案

### 検証方法

人間による方針決定（AI 委任境界外：研究分野の最終選定は人間が判断）。
