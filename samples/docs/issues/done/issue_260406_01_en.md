## I1. Verify n-Invariance of Family I/II Certificates

### Background

According to `docs/theory/triviality_certificates.md`, the triviality certificates
for Families I and II satisfy an n-invariance property: the certificate vector
is independent of the algebra rank n for n ≥ n₀.

The code repository ([osp-triviality](https://github.com/ritsumei-aoi/osp-triviality))
provides `scripts/verify_n_invariance.py` for checking this property.
If the n-invariance holds, a single certificate verified for small n
proves the result for all n, which is a key step in the triviality proof.

### Requirements

1. Read `docs/theory/triviality_certificates.md` and confirm the n-invariance claim
2. Run `scripts/verify_n_invariance.py` from the osp-triviality repository
   for B(0,1) through B(0,3) and confirm the certificates match across ranks
3. Summarize which families and weight types are covered

### Completion Criteria

- [x] The n-invariance property is confirmed for Family I
- [x] The n-invariance property is confirmed for Family II
- [x] Results for B(0,1), B(0,2), B(0,3) are compared and reported
