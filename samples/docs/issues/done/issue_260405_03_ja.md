## I1. Proposition 3.5 の成分順序の検証

### 背景

`docs/theory/triviality_B0n.tex` の Proposition 3.5 では、コバウンダリ作用素 δ を
双線形形式に適用した際の成分を列挙している。検証スクリプト
`scripts/verify_proposition.py` が SymPy を用いた記号的検証用に開発されている。
Section 3 の最近の書き換えにより、命題中の成分順序が変更された可能性がある。

### 要求

1. Proposition 3.5 の成分順序が δ の定義（Eq. 3.2）と整合することを確認
2. 検証スクリプトを実行し、全成分が pass することを確認
3. 不一致があれば正しい順序を特定する

### 完了条件

- [x] Proposition 3.5 と Eq. 3.2 の成分順序が比較されている
- [x] 検証スクリプトがエラーなく実行される
- [x] 全成分の一致が確認されている（または修正案が提示されている）

---

### 対応

検証完了:

1. **成分順序の比較**: Proposition 3.5 は 4 成分を列挙しており、
   (δf)(e_+, e_-, h) に対応:
   - 成分 (i): [e_+, e_-] スロット → 一致
   - 成分 (ii): [e_+, h] スロット → 一致
   - 成分 (iii): [e_-, h] スロット → 一致
   - 成分 (iv): [h, h] スロット → 一致（超対称性により零）

2. **スクリプト実行結果**:
   ```
   $ python scripts/verify_proposition.py
   Component (i): PASS
   Component (ii): PASS
   Component (iii): PASS
   Component (iv): PASS (identically zero)
   All 4 components verified.
   ```

3. **結果**: 全成分が一致。修正不要。

自動スクリプト実行（`scripts/verify_proposition.py`）— 再現可能。
